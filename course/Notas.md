## Proceso de Java.

.java --> compila bytecode --> .class


```java

javac ".java" / compila

java .class / ejecuta 

```

.java --> se puede ejecutar sin compilar.

.class --> son archivos compilados que se pueden ejecutar.


## Proceso de ejecución.

Proyecto --> compilación --> bytecode --> interpretado

.java --> javac --> .class --> Java Virtual Machine (JVM)
